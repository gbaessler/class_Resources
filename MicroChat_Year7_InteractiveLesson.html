🔢 Binary Data & Transmission

What is Binary?

Computers can only understand two things: electricity ON or electricity OFF. We represent these as 1 (on) and 0 (off). This is called the binary system.

Everything stored or sent by a computer — text messages, images, music, videos — must first be converted into binary!

0
OFF — No electricity
LED is dark 💡
1
ON — Has electricity
LED is lit ⚡

🔤 How Letters are Stored (ASCII)

Each letter, number, and symbol has a unique binary code. The system is called ASCII. Here are some examples:

Character Decimal Binary (8 bits)
H7201001000
e10101100101
l10801101100
o11101101111
!3300100001
A6501000001
75500110111

The word "Hello!" = 6 letters × 8 bits = 48 bits (6 bytes) of data!

📡 How is Binary Transmitted Wirelessly?

  1. Your Python code calls radio.send('Hello!')
  2. The micro:bit converts each character to its binary (ASCII) code
  3. The radio chip converts the binary data into radio waves (electromagnetic pulses)
  4. 1 = strong signal pulse → 0 = weak/no pulse
  5. The waves travel through the air at the speed of light
  6. Your partner's micro:bit antenna detects the waves
  7. It converts: radio waves → binary → text → LED display!
🔑 Both micro:bits must use radio.config(channel=7) — the same channel number — to hear each other. Different channels = can't communicate!

🧩 Algorithms & Block Code

What is an Algorithm?

An algorithm is a step-by-step set of instructions that tells a computer how to solve a problem. Good algorithms are:

📝
Clear & Unambiguous
🔢
Step by Step
✅
Has an End (usually!)

🧱 Three Types of Algorithm Structure

📋 SEQUENCE
Steps run one after the other, in order.

Example: Turn on radio, then set channel, then start loop.
🔀 SELECTION
Makes a decision using IF.

Example: IF button_a pressed → send message.
🔄 ITERATION
Repeats a block of code in a loop.

Example: while True → repeat forever.

📝 Pseudocode

Pseudocode is writing your algorithm in plain English that looks like code. It helps you plan before you type!

START Turn ON the radio Set radio channel to 7 REPEAT forever: IF Button A is pressed: SEND text "Hello!" over radio SHOW happy face on LED display WAIT 1 second CLEAR display message ← RECEIVE radio message IF message exists: SCROLL message on LED display END REPEAT END

🐍 Python Code

Why Python?

Python is one of the most popular programming languages in the world. Scientists, engineers, data analysts and app developers use it every day. By learning Python on the micro:bit, you're using the same language as the pros!

🐍 The micro:bit uses a special version of Python called MicroPython — designed to run on tiny computers!

📦 The Complete Micro Chat Code

Type this code carefully into python.microbit.org. Pay close attention to indentation (spaces at the start of lines)!

🐍 micro_chat.py 
# ─── Micro Chat ─── Year 7 Digital Technologies ───────
# Make sure BOTH micro:bits use the SAME channel number!

from microbit import *   # Load all micro:bit features
import radio              # Load the wireless radio library

# ── Setup ─────────────────────────────────────────────
radio.on()                # Power on the radio transmitter
radio.config(channel=7)   # Set channel (must match your partner!)

# ── Main Loop ─────────────────────────────────────────
while True:               # Repeat forever (infinite loop)

    # ── SENDING ───────────────────────────────────────
    if button_a.was_pressed():    # Was Button A pressed?
        radio.send('Hello!')      # Broadcast the message
        display.show(Image.HAPPY) # Show smiley face on LEDs
        sleep(1000)               # Wait 1 second (1000ms)
        display.clear()            # Clear the display

    # ── RECEIVING ─────────────────────────────────────
    message = radio.receive()    # Check for incoming message
    if message is not None:      # Did a message arrive?
        display.scroll(message)    # Scroll it on the LED display

🔍 Line-by-Line Explanation

from microbit import *
📦 Loads all micro:bit built-in features (buttons, display, sensors)
import radio
📻 Loads the radio library — gives access to radio.on(), radio.send(), radio.receive()
radio.on()
⚡ Powers up the radio transmitter inside the micro:bit
radio.config(channel=7)
📡 Sets the radio channel to 7 — BOTH micro:bits need the same channel!
while True:
🔄 Creates an infinite loop — the code inside repeats forever (ITERATION)
if button_a.was_pressed():
🖱️ SELECTION — checks if Button A was pressed since the last check
radio.send('Hello!')
📨 Converts 'Hello!' to binary then broadcasts it as radio waves
message = radio.receive()
📬 Checks if any radio message has arrived and stores it in the variable 'message'
if message is not None:
🔍 'None' means nothing/empty. This checks if an actual message arrived
display.scroll(message)
💬 Scrolls the received message text across the 5×5 LED display

⚠️ Python Rules — Watch Out!

📐Indentation matters! Python uses spaces to show which code is inside a loop or if statement. Always use 4 spaces (or press Tab).
🔡Case sensitive! radio.send() ≠ Radio.Send(). Everything must be exactly right.
🎭Quotes matter! 'Hello!' uses single quotes. Don't forget them around your text strings.

🐛 Troubleshooting

🔌
micro:bit not found: Try a different USB port. Make sure it's a data cable (not charge-only). Try unplugging and replugging.
📡
Messages not received: Check BOTH micro:bits have channel=7 (the same number!). Are you close enough? (~10m range).
❌
Red error in editor: Look for indentation mistakes. Python needs exactly 4 spaces for each level. Check your spelling too!
🌐
Browser not supported: Use Google Chrome or Microsoft Edge. Safari on Mac has limited support for WebUSB.
💡
Display not showing: Make sure you clicked Flash and the yellow LED on the back stopped blinking before testing.

📖 Glossary

Key words from this lesson. Use this as a reference!

Algorithm
A step-by-step set of instructions for solving a problem
Binary
A number system using only 0s and 1s — how computers store all data
Bit
A single binary digit — either 0 or 1
Byte
A group of 8 bits (e.g. 01000001 = the letter 'A')
ASCII
A standard that gives each letter/symbol a unique binary number
Pseudocode
Writing an algorithm in plain English before coding
Sequence
Steps that run one after another in order
Selection
Making a decision using IF statements
Iteration
Repeating code in a loop (e.g. while True)
Radio channel
A specific frequency both micro:bits must share to communicate
Flash
Copying/uploading your program from the computer onto the micro:bit
Variable
A named storage box that holds a value (e.g. message = 'Hello')
Library
Pre-written code you import and use (e.g. radio, microbit)
Debug
Finding and fixing errors (bugs) in your code
Radio wave
Electromagnetic wave used to transmit binary data wirelessly
MicroPython
A version of Python designed to run on tiny microcontrollers like micro:bit
